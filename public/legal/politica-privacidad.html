Este correo se utiliza exclusivamente para comunicaciones legales, privacidad y ejercicio de derechos. Para ayuda sobre el servicio, el canal principal es el bot de TuCita.

2. Datos tratados desde la web

La web de TuCita es estática. Desde esta landing no se solicitan formularios, pagos, documentos ni datos personales sensibles.

  • Navegación técnica básica necesaria para servir la web.
  • No hay formularios web.
  • No hay pagos dentro de la web.
  • No se reciben documentos desde la web.

3. Datos tratados desde el bot

Cuando el usuario decide iniciar el flujo dentro del bot, pueden tratarse datos como:

  • Telegram ID.
  • Nombre del titular.
  • Tipo y número de documento.
  • Nacionalidad.
  • Datos necesarios para configurar o reservar una cita.
  • Documento o scan solo si el flujo autorizado lo requiere.
  • Pagos o estado del plan.
  • Registros de auditoría y seguridad.

4. Finalidades

  • Prestar el servicio contratado.
  • Configurar la vigilancia de citas.
  • Gestionar la autorización aceptada por el propio titular.
  • Procesar pagos mediante Stripe.
  • Reservar cita si corresponde y existe disponibilidad oficial.
  • Atender incidencias dentro del bot.
  • Cumplir obligaciones legales, de seguridad o auditoría.

5. Base jurídica

  • Ejecución del contrato o servicio solicitado.
  • Consentimiento y autorización del usuario titular.
  • Cumplimiento de obligaciones legales.
  • Interés legítimo en seguridad, prevención de abuso y auditoría, si aplica.

6. Encargados y proveedores

Para prestar el servicio pueden intervenir proveedores como:

  • Telegram, como canal de interacción con el bot.
  • Stripe, para procesamiento de pagos.
  • Railway, para infraestructura del bot.
  • GitHub Pages, para la web estática.
  • PostgreSQL/Railway, si aplica al almacenamiento del bot.

No vendemos datos personales.

7. Documentos y canales informales

Por seguridad, no se reciben documentos ni datos sensibles fuera del flujo autorizado del bot. No deben enviarse documentos por correo, redes sociales u otros canales no autorizados.

8. Derechos

El usuario puede ejercer sus derechos de acceso, rectificación, supresión, oposición, limitación, portabilidad y revocación cuando proceda. La revocación operativa del servicio puede solicitarse mediante el comando /revocar dentro del bot.

9. Conservación

Los datos se conservarán mientras dure el servicio, hasta la revocación cuando proceda, y durante los plazos mínimos necesarios para obligaciones legales, seguridad o auditoría.

10. Seguridad

TuCita aplica medidas de seguridad como cifrado, acceso limitado y registros de auditoría. Ninguna medida técnica elimina todos los riesgos, pero se trabaja con el principio de minimización y control de acceso.

11. Reclamaciones

Si consideras que el tratamiento no se ajusta a la normativa, puedes presentar una reclamación ante la Agencia Española de Protección de Datos (AEPD).
